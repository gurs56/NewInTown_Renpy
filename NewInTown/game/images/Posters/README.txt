POSTERS  (one-shot story paintings / CG)
=========================================

Drop finished poster art in THIS folder as a .png whose filename
EXACTLY matches the poster's canonical name used in the script.
The name is load-bearing: the game shows a placeholder card with
that name until a file of the same name appears here.

Format: PNG, 1920 x 1080.

Helper / how it's used:  game/Scripts/Posters/Posters.rpy
    scene expression Poster("A00 - Intercity_Bus") with dissolve

-------------------------------------------------------------
A00 - INTRO (Bus Terminal) needs these files:
-------------------------------------------------------------
    A00 - Intercity_Bus.png
    A00 - Mothers_Hands.png
    A00 - Old_Photo.png
    A00 - Bus_City.png
    A00 - Step_off.png
